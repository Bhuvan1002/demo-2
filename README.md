This is a simple QR Code Generator project created by a beginner while learning Python and its basic libraries.

This project helped me understand:

Python functions

User input

Working with external libraries

File and folder handling in Python

Requirements
pip install qrcode[pil]

How to Run
python main.py


Enter a URL when prompted. The QR code will be saved in the qrcodes folder.

Learning Purpose

This project helped me practice Python basics, using libraries, and file handling.
